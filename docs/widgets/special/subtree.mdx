# Subtree

| Название типа| Поддерживает анимации свойств | Контроллируемый по умолчанию | Может содержать связанное действие | Ссылка |
| - | - | - | - | - |
| Subtree | &cross;  | :heavy_check_mark: | &cross; | - |

Виджет `Subtree` используется для управления поддеревьями виджетов. 
Атрибуты виджета могут хранить json, который представляет собой новое или измененное поддерево виджетов. 
Это позволяет изолировать поддерево виджетов и управлять им через обновление корневого элемента, динамически меняя контент `Subtree`.
