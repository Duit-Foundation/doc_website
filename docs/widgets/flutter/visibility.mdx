# Visibility

| Название типа| Поддерживает анимации свойств | Контроллируемый по умолчанию | Может содержать связанное действие | Ссылка |
| - | - | - | - | - |
| Visibility | &cross; | &cross; | &cross; | [link](https://api.flutter.dev/flutter/widgets/Visibility-class.html) |

:::info
Виджет принимает до двух дочерних элементов. Первый — основной виджет, второй (опциональный) — виджет-замена, отображаемый при `visible: false`. Если замена не задана, используется `SizedBox.shrink()`.
:::
