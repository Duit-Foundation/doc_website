# ClipOval

| Название типа| Поддерживает анимации свойств | Контроллируемый по умолчанию | Может содержать связанное действие | Ссылка |

| ClipOval | &cross; | &cross; | &cross; | [link](https://api.flutter.dev/flutter/widgets/ClipOval-class.html) |

Виджет, который обрезает дочерний элемент по овалу (или кругу, если ширина и высота равны).

## Атрибуты

| Атрибут | Тип | Описание |
| - | - | - |
| clipBehavior | Clip | Поведение обрезки (по умолчанию: antiAlias) |

### Значения clipBehavior

- `none` — без обрезки
- `hardEdge` — жёсткая обрезка без сглаживания
- `antiAlias` — обрезка со сглаживанием (по умолчанию)
- `antiAliasWithSaveLayer` — обрезка со сглаживанием и сохранением слоя
