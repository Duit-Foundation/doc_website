# Switch

| Switch | &cross;  | :heavy_check_mark: | :heavy_check_mark:| [link](https://api.flutter.dev/flutter/material/Switch-class.html) |

:::info
Значение, хранимое виджетом, может быть использовано в качестве источника данных для действий.

```dart
final class SwitchAttributes extends AttendedModel<bool>
    implements DuitAttributes<SwitchAttributes> {
  final Color? activeColor,
      focusColor,
      hoverColor,
      activeTrackColor,
      inactiveTrackColor;
  final WidgetStateProperty<Color>? overlayColor,
      trackColor,
      thumbColor,
      trackOutlineColor;
  final WidgetStateProperty<double>? trackOutlineWidth;
  final double? splashRadius;
  final MaterialTapTargetSize? materialTapTargetSize;
  final bool? autofocus;

  SwitchAttributes({
    required super.value,
    this.activeColor,
    this.focusColor,
    this.hoverColor,
    this.activeTrackColor,
    this.inactiveTrackColor,
    this.overlayColor,
    this.trackColor,
    this.thumbColor,
    this.trackOutlineColor,
    this.trackOutlineWidth,
    this.splashRadius,
    this.materialTapTargetSize,
    this.autofocus,
  });

```
