# WebSocketTransportOptions

:::danger Deprecated
Класс `WebSocketTransportOptions` помечен как deprecated и будет удален в следующем мажорном релизе.

Используйте [WSTransportManager](/docs/api/dart_api/WSTransportManager) вместо него.
:::

Класс для конфигурации встоенной реализации WebSocket-транспорта. Имплементирует интерфейс [Streamer](/docs/api/dart_api/Streamer) и обрабатывается драйвером, как потоковый источник данных.
