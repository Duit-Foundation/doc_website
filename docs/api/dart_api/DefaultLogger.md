# DefaultLogger

:::danger Deprecated
Класс `DefaultLogger` помечен как deprecated и будет удален в следующем мажорном релизе.

Используйте [LoggingCapabilityDelegate](/docs/api/dart_api/LoggingCapabilityDelegate) вместо него.
:::

Базовая реализация интерфейса [DebugLogger](/docs/api/dart_api/DebugLogger), логирует сообщения в консоль в debug режиме.
