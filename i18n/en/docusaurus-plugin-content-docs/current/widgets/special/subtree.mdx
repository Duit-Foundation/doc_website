# Subtree

| Type name | Supports animations | Controlled by default | May contain associated action | Link |
| - | - | - | - | - |
| Subtree | &cross;  | :heavy_check_mark: | &cross; | - |

The `Subtree` widget is used to manage widget subtrees. 
Widget attributes can store json, which represents a new or changed widget subtree. 
This allows you to isolate a widget subtree and manage it by updating the root element, dynamically changing the content of the `Subtree`.

## Attributes

```dart
final class SubtreeAttributes implements DuitAttributes<SubtreeAttributes> {
  final Map<String, dynamic>? data;

  SubtreeAttributes({
    this.data,
  });
```
