# Meta

| Type name | Supports animations | Controlled by default | May contain associated action | Link |
| - | - | - | - | - |
| Meta | &cross; | :heavy_check_mark: | &cross; | - |

InheritedWidget that stores json containing meta information for the Duit view.

:::info
The value stored by the widget can be used as a data source for actions.

[More](/docs/core_concepts/actions_events)
:::

The widget can be useful in cases:
- When it is necessary to transmit additional information about the UI element in the layout, which can be used during the execution of actions.
- When it is necessary to store an intermediate state

## Attributes

```dart
final class MetaAttributes extends AttendedModel<Map<String, dynamic>>
    implements DuitAttributes<MetaAttributes> {

  MetaAttributes({
    required super.value,
  });
```
